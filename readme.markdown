# INSTALL 安装

* Install expect 使用如下命令，先安装expect包

    Red Hat: `yum install expect`

    Debian:  `apt-get install expect`

* 在服务器上测试ssh链接备份机

* Run install.sh 运行install.sh脚本

    `chmod +x ./install.sh`
    
    `./install.sh`

# uninstall 卸载

* Run uninstall.sh 运行uninstall.sh脚本

    `./uninstall.sh`
